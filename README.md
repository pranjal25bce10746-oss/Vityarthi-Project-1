# Python Bank Application

## Overview

This project implements a simple command-line banking system in Python.
It allows users to create accounts, deposit money, withdraw funds, view
account details, and close accounts.

## Features

-   Create a new bank account
-   Generate unique 10-digit account numbers
-   Deposit and withdraw money
-   View balance and account details
-   Close existing accounts
-   Interactive text-based user interface

## Technologies Used

-   Python 3
-   Object-Oriented Programming (OOP)

## How to Run

1.  Ensure you have Python installed on your system.
2.  Save the script as `bank.py`.
3.  Open a terminal in the script directory.
4.  Run the following command:

```{=html}
<!-- -->
```
    python bank.py
